# test-tdd

## How it was built
- Installated corepack `npm -i -g corepack`
- Initialized yarn `yarn init`
- Added axios, express, body-parser `yarn add axios express body-parser`
- Added jest `yarn add -D jest`
- Coded API
- Coded tests

## How to test
- Execute `yarn test`
